# Image Processing Application
This project is a text-based image processing program written in C. 
It allows users to load a PGM image, apply filters, perform edge detection, 
resize images, and compute LBP features.

# Features
- Zoom / Shrink Image
- Average, Mean, Median Filters
- Edge Detection (Canny, Sobel, Prewitt)
- LBP computation
- Save processed image

# How to Run
A sample PGM file (`test.pgm`) is included for testing the program.
1. Compile the code:
   gcc main.c -o program
2. Run:
   ./program
